# Documentation Website

2025 트렌드를 반영한 현대적인 디자인의 MD 문서 웹사이트입니다.

## 기술 스택

- **Vite** - 빠른 개발 서버 및 빌드 도구
- **React 18** - UI 라이브러리
- **TypeScript** - 타입 안정성
- **Tailwind CSS** - 유틸리티 기반 CSS 프레임워크
- **React Router** - 클라이언트 사이드 라우팅
- **React Markdown** - Markdown 렌더링
- **Lucide React** - 현대적인 아이콘 라이브러리

## 설치

\`\`\`bash
npm install
\`\`\`

## 개발 서버 실행

\`\`\`bash
npm run dev
\`\`\`

개발 서버는 http://localhost:3000 에서 실행됩니다.

## 빌드

\`\`\`bash
npm run build
\`\`\`

## 미리보기

\`\`\`bash
npm run preview
\`\`\`

## 문서 추가하기

\`docs/\` 폴더에 새로운 Markdown 파일을 추가하고, \`src/App.tsx\`의 \`docsList\` 배열에 경로와 파일명을 추가하면 됩니다.
